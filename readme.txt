###Framework for the Unification of SR, OLPA and ARM. A sample readme file to install the code for the framework for the Unification of SR, OLAP and ARM.
##............Installation...................##
Configure Internet Information System (IIS) in a Windows system.
Download the code and Host it in the Localhost
Test the connection weather the application is working or not
Use the dataset provided in the folder for generate the All combinations of input factors(ACIF) and different operations of SR, OLAP and ARM
Datasets
Two sample datasets UDS1.XLS and UDS2.XLS are given in the repository, however, an  Orcale database can also be configured with the application
